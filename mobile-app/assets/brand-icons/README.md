# Fuel Path Brand Icons

These icons are local copies of favicons or app icons discovered from official
brand websites for prototype validation. They are used to test recognisable map
pins and result rows, not to claim licensed brand artwork for a public product.

Budget uses the three-colour diamond cropped from the public Budget Petrol /
Gascorp site logo assets.

The expanded icon pass adds official-site favicon/logo-derived prototype assets
for Shell, Speedway, Pearl Energy, U-GO, Enhance, ARKO, Liberty, Ultra, APW,
Inland Petroleum, APCO and ASTRON. Independent and generic-fuel are neutral
prototype marks for FuelCheck brands without a dependable public logo source.

Before any public or commercial release, confirm each brand's trademark and logo
usage requirements or replace these files with licensed assets supplied through
brand/media request channels.
